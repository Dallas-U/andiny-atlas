# Andiny Atlas Engineering Handbook

Welcome to the engineering handbook for **Andiny Atlas**.

This handbook documents the architecture, engineering practices, design decisions, debugging strategies, testing philosophy, deployment process, and sprint history of the project.

It exists to answer an important question:

> **"Why was the system built this way?"**

Rather than documenting only the finished code, this handbook captures the reasoning behind the implementation.

Every sprint contributes to both the application and this handbook.
